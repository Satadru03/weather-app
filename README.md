# Weather Displaying Program

A temperature displaying program that retrieves weather data using APIs and displays the current temperature along with a text-to-speech output.

## Installation

1. Clone or download this repository to your local machine.

2. Install Required Python Modules:
   - Open a terminal and navigate to your project directory.
   - Run the following command to install the required modules:
     ```
     pip install requests json os sys
     ```

3. Obtain API Key:
   - You need an API key to access weather data from a service like OpenWeatherMap.

## Usage

1. Navigate to the project directory using a terminal.

2. Run the temperature displaying program: main.py

3. Follow the on-screen instructions to input the location for which you want to fetch the weather data.

4. The program will retrieve the temperature data using the API and display it along with a spoken output.

## Features

- Retrieve weather data using APIs and display the current temperature.
- Utilize the text-to-speech functionality to read out the temperature.
